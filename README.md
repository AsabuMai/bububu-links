# bububu-links

Vue + Vite personal links site.

## Local development

```bash
npm install
npm run dev
```

## GitHub Pages

This repo is configured to deploy automatically from the `main` branch with GitHub Actions.

After pushing:

1. Open the repository on GitHub
2. Go to `Settings` -> `Pages`
3. Set `Source` to `GitHub Actions`

Then GitHub will publish the site automatically.
